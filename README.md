# Aetherium Desktop

Desktop application for Aetherium Chat, built with Electron.

## Features

- Native desktop experience
- System tray integration
- Auto-updates
- Minimizes to tray
- All features of the web app

## Development

### Prerequisites

- Node.js 18+
- npm or yarn

### Setup

```bash
npm install
```

### Run in development

```bash
npm start
```

### Build

```bash
# Windows
npm run build:win

# macOS
npm run build:mac

# Linux
npm run build:linux
```
