# Aetherium Desktop

Desktop application for Aetherium Chat, built with Electron.

---

## ⚠️ Important Update (v1.1.0)

**If you have a version of Aetherium Desktop from before February 5, 2026**, please uninstall it and download the latest version. The old version had an incorrect server URL and will not connect properly.

**To update:**
1. Uninstall the old Aetherium app
2. Download the new installer from [Releases](https://github.com/TheEtherWolf/aetherium-desktop/releases/latest)
3. Run the installer

---

## Features

- Native desktop experience
- System tray integration
- Auto-updates
- Minimizes to tray
- All features of the web app

## Development

### Prerequisites

- Node.js 18+
- npm or yarn

### Setup

```bash
npm install
```

### Run in development

```bash
npm start
```

### Build

```bash
# Windows
npm run build:win

# macOS
npm run build:mac

# Linux
npm run build:linux
```
